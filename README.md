# Sauerteig Manager (Sourdough manager)

Eine benutzerdefinierte Home Assistant Integration zur komfortablen Verwaltung und Protokollierung deiner Sauerteige.

Mit dieser Integration kannst du direkt über ein interaktives Dashboard protokollieren, wann du deinen Sauerteig das letzte Mal gefüttert hast. Sie erstellt pro Sauerteig ein eigenes Gerät, das einen Füttern-Button sowie einen Sensor für den exakten Zeitstempel der letzten Fütterung beinhaltet.

## Features
- **Multi-Sauerteig-Support:** Füge beliebig viele Sauerteige bequem über die Home Assistant Benutzeroberfläche hinzu.
- **Interaktiver Füttern-Button:** Ein Knopfdruck registriert die Fütterung sofort.
- **Live-Sensor (`timestamp`):** Zeigt in Echtzeit an, wann die letzte Fütterung stattfand.
